== Description ==

Filter Featured Image can help you filter you post with or without featured image.

= Usage =
1. install plugin from wordpress plugin directory
2. enter Posts Page
3. use Dropdown to switch filter

== Frequently Asked Questions ==

= What's mean about difference options? =

- Show All Posts: show all Posts with or without featured image.
- Show Post With Featured Image: only show posts which have featured image.
- Show Post Without Featured Image: only show posts which don't have featured image.
